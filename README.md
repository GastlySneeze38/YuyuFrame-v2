# YuyuFrame - Launcher Minecraft Premium

## À propos

YuyuFrame est un launcher Minecraft premium offrant une expérience de jeu optimisée avec synchronisation cloud, gestion avancée des instances et des fonctionnalités exclusives pour les joueurs exigeants.

## Plans d'abonnement

### Plan Free - 0€/mois
- Launcher Minecraft complet
- Instances locales illimitées
- 2 comptes Minecraft
- Gestion des mods
- Console de jeu intégrée

### Plan Premium - 7.99€/mois
- Tout ce qui est inclus dans Free
- Sync cloud (jusqu'à 3 saves)
- Jusqu'à 4 instances synchronisées
- Restauration cloud en un clic
- Support prioritaire

### Plan Ultimate - 15.99€/mois
- Tout ce qui est inclus dans Premium
- Sync cloud (jusqu'à 10 saves)
- Jusqu'à 10 instances synchronisées
- Restauration cloud en un clic
- Support prioritaire
- Accès anticipé aux nouvelles fonctionnalités

## Fonctionnalités

- Installation automatique de Minecraft et des mods
- Gestion simplifiée des modpacks et instances
- Synchronisation cloud des configurations et sauvegardes
- Console de jeu intégrée
- Mises à jour automatiques du launcher
- Interface moderne et intuitive
